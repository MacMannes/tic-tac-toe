# tic-tac-toe

# Prerequisites

You have to have node installed.

## Start a game

You can start a game by running the following commands:

```shell
npm install
npm run start
```

## Unit tests

Run the unit test with this command:

```shell
npm run test
```

## Coverage of unit tests

You can check the code coverage and view the html results by running this command:

```shell
npm run coverage && open coverage/index.html

```
